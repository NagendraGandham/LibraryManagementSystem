# Simple Library Management System (Java)

##  Overview

A simple Java project demonstrating **OOP concepts**
like inheritance, polymorphism, and encapsulation.
Users can borrow and return **Books** and **Magazines**.

##  Features

* 📖 Books and 📰 Magazines
* 🎓 Students and 👨‍🏫 Professors
* Borrow & return system
* Item availability tracking


##  Constraints

* A user can borrow only **one item at a time**
* Each item has only **one copy**
* Students can borrow only **Books**
* Professors can borrow **Books & Magazines**

##  Concepts Used

* Inheritance
* Method Overriding
* Encapsulation
